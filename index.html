MoeSkin Marketplace

社区创作者所制作的用于萌娘百科 MoeSkin 皮肤的 CSS、JS。

通用嵌入方法

CSS

  • （推荐）如果你的common.css内不存在任何生效的内容（单纯的空白页面或注释不影响），可选择在moeskin.css（的前面）插入引用代码；
  • （否则）在common.css（的前面）插入引用代码。

JS

  • （推荐）如果你的common.js内不存在任何生效的内容（单纯的空白页面或注释不影响），可选择在moeskin.js（的前面）插入引用代码；
  • （否则）在common.js（的前面）用以下代码包裹引用代码。 
    if (mw.config.get().skin === 'moeskin') {
	// 引用代码	
}

当然你也可以选择手动复制代码到上述页面来手动控制具体内容。
